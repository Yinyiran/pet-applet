# 2026-06-08 工作日志

## 会员权益体系更新

根据新的会员等级规则修改了会员权益弹窗和相关逻辑：

### 修改文件
- `index.html` — member-detail-sheet 三级卡片内容更新
- `js/profile.js` — 等级门槛、updateMemberLevel、openMemberDetail 高亮逻辑
- `js/points.js` — 积分倍率 memberMultiplier
- `style.css` — 新增 .member-level-card.active 高亮样式

### 新规则
| 等级 | 门槛 | 折扣 | 积分倍率 | 生日权益 |
|------|------|------|---------|---------|
| 银牌 | ¥500+ | 9.8折 | 1.1x | 一份 |
| 金牌 | ¥1,000+ | 9.6折 | 1.3x | 一份（2年） |
| 黑钻 | ¥3,000+ | 9.3折 | 1.5x | 一份（终身） |

### 关键变更
- 黑钻门槛：¥5,000 → ¥3,000
- 移除了金牌和黑钻的额外权益（专属客服、新品优先体验等），仅保留折扣+积分+生日三项核心权益
- openMemberDetail() 打开弹窗时自动高亮当前等级卡片（橙色边框+阴影+右上角"当前等级"标签）

## 购物车收货地址功能

在购物车页面顶部新增收货地址栏，完整的地址选择/管理功能：

### 修改文件
- `index.html` — cartPage 内 cartContent 顶部新增 `.cart-address-bar` 地址栏 HTML
- `style.css` — 新增 `.cart-address-bar` 全套样式 + `.address-popup-card.selected` 选中状态样式
- `js/cart.js` — 新增 `cartSelectedAddressId` 状态变量、`renderCartAddress()` 渲染函数、`openCartAddressList()` 打开选择器、`selectCartAddress(id)` 选中回调
- `js/profile.js` — `renderAddressListPopupBody()` 支持购物车选择模式（`_cartAddressMode`）、`closeAddressListPopup()` 重置模式、`saveAddress()` 和 `deleteAddressInPopup()` 同步刷新购物车地址栏

### 功能要点
- 地址栏位于购物车商品列表上方，白色卡片样式，显示当前选中地址或"请选择收货地址"
- 首次进入自动选择默认地址（无默认则选第一个）
- 点击地址栏弹出地址列表，标题变为"选择收货地址"
- 列表模式下点击地址卡片即选中（带橙色边框+勾选标记），操作按钮（编辑/删除/设默认）通过 `stopPropagation` 保持可用
- 支持在弹窗内新增、编辑、删除地址，操作后自动同步购物车地址栏
- 删除地址时若该地址为当前选中，自动清除选择
