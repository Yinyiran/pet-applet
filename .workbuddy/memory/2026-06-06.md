## 同城卡片 - 导航和电话功能

**需求（v1）**：在 city-card 的 `.city-card-addr` 行右侧增加导航图标和电话文字。

**需求（v2）**：导航图标放到 `.city-card-addr` 地址后面，电话放到 `.city-card-tags` 评分后面（图标+号码）。

### 修改内容

1. **cityData 数据**：为每个商家添加 `tel` 字段（电话号码）

2. **JS (`js/city.js`)**：
   - `.city-card-tags` 恢复简单标签布局
   - `.city-card-addr` 改为两段式：左侧地址（SVG定位图标 + 文字），右侧导航图标 + 电话文字
   - 导航使用 `navigateTo(idx)` 函数，调用高德地图 URI Scheme 按地址搜索
   - 电话使用 `copyTel(tel)` 函数，优先 Clipboard API，降级用 `execCommand('copy')`
   - 两个操作都通过 `event.stopPropagation()` 阻止冒泡，不触发卡片点击

3. **CSS (`style.css`)**：
   - `.city-card-tags` 恢复原始 flex-wrap 布局
   - 删除之前添加的 `.city-card-tags-left/right`、`.city-card-action` 样式
   - `.city-card-addr` 改为 `justify-content: space-between`
   - 新增 `.city-card-addr-left`（左侧，flex:1 + 文字截断）和 `.city-card-addr-right`（右侧，flex-shrink:0）
   - 新增 `.city-card-nav`（导航按钮，22px圆形，主色）和 `.city-card-tel`（电话文字，主色加粗，点击复制）
