     - `.poster-preview-overlay`：fixed 全屏遮罩，z-index 1000
     - `.poster-preview-card`：白色圆角卡片，max-width 380px，max-height 88vh，slideUp 入场动画
     - `.poster-preview-header`：标题 + 圆形关闭按钮
     - `.poster-preview-body`：灰色背景，居中展示图片
     - `.poster-preview-actions`：两个按钮（outline + primary）

3. **`js/quiz.js`**（新增 5 个函数）：
   - `generatePosterImage()`：
     - 检查 html2canvas 是否加载
     - 临时隐藏底部栏 → 滚动到顶部 → 调用 html2canvas（scale:2 高清输出）
     - 恢复底部栏 → 将 canvas 转为 dataUrl → 填充到预览 img → 显示弹窗
   - `closePosterPreview(e)`：点击遮罩关闭弹窗（子元素冒泡不触发），清空 img.src
   - `dataUrlToBlob(dataUrl)`：辅助函数，将 base64 dataUrl 转为 Blob 对象
   - `downloadPosterImage()`：
     - 创建 a 标签，设置 download 属性和 dataUrl href
     - 临时添加到 body → click() → 移除（兼容所有浏览器）
     - Toast 提示「已保存海报图片」
   - `sharePoster()`（底部「分享」按钮点击）：
     - 如果图片未生成 → 调用 `generatePosterImage()`
     - 如果已生成 → 直接显示预览弹窗
   - `sharePosterImage()`（预览弹窗内「分享给好友」点击）：
     - 用 `dataUrlToBlob()` 将图片转为 Blob → 封装为 File 对象
     - 优先使用 `navigator.canShare({ files })` + `navigator.share({ files })` 分享图片文件
     - 不支持文件分享时降级到 `navigator.share({ url })` 分享链接
     - 完全不支持 Web Share API 时 Toast 提示「请长按海报图片分享」

### 技术细节

- **html2canvas 配置**：scale:2（高清）、useCORS:true、backgroundColor 匹配页面暖米色
- **Web Share API 降级策略**：文件分享 → 链接分享 → 提示长按保存
- **dataUrl 判断**：用 `startsWith('data:image')` 判断是否已生成海报图片（空字符串返回 false）
- **下载兼容**：a 标签必须临时添加到 DOM 并触发 click，否则某些浏览器不会下载
